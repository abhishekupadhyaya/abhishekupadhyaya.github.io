AWS Data Pipeline

I am currently working on this project for Pearson Lab at Duke University. I am using Amazon Web Services tools such as S3, Lambda, EC2, RDS, and DynamoDB to store, process, and analyze large amount of data. To support the pipeline, I have designed a web application using Python Flask to query and download the data from S3. I have also written Python scripts to automate the processes of restoring DB snapshots as well as starting, stopping EC2 instances

The code to upload the files to S3 including the python scripts used in AWS Lambda to redirect meta-data to RDS and DynamoDB are in this GitHub repository. Similarly, the code to query the database and download the files from AWS S3 are in this GitHub repository.
